# CPU_noint — 无中断硬件平台

## 概述

`CPU_noint` 是最简的 MicroBlaze 硬件平台，**不含中断控制器（INTC）和 AXI Timer**。仅通过 AXI GPIO 以**纯轮询（Polling）** 方式读取外设状态并输出控制信号。适用于简单的 GPIO 测试和无需中断响应的应用场景。

---

## 硬件配置

| 外设 | 功能 | 通道1 | 通道2 |
|------|------|-------|-------|
| **AXI GPIO 0** | 16 位拨码开关输入 + 16 位 LED 输出 | 开关（输入，16位） | LED（输出，16位） |
| **AXI GPIO 1** | 8 位数码管位选 + 8 位数码管段码输出 | 位选（输出，8位） | 段码（输出，8位） |
| **AXI GPIO 2** | 5 位按键输入 | 按键（输入，5位） | — |

> ⚠ **不含 INTC、不含 AXI Timer**，无法使用中断方式和定时器功能。

## 硬件特性

- **中断控制器**：❌ 无
- **定时器**：❌ 无
- **驱动方式**：仅支持**轮询（Polling）**
- **适用处理器**：MicroBlaze

## 使用场景

- 学习轮询方式驱动外设的基础原理
- 不支持中断的开发板或简化硬件环境
- 无需实时响应、逻辑简单的 GPIO 控制
