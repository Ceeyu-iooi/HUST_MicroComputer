# Test1 — LED实时 + 全部数码管显示按键字符

## 题目描述
拨动16位开关，LED灯实时显示开关状态。每按一次C/U/L/R/D键，所有（8个）数码管全部显示对应字符（C、U、L、R、D）的段码。

## 硬件平台
CPU_INT_TIMER

## 中断设备
- GPIO_0 通道1（16位输入，开关）：开关中断
- GPIO_1（8位输出，位码 + 段码）：数码管动态扫描
- GPIO_2 通道1（5位输入，按键）：按键中断
- Timer_0 T0：数码管动态扫描定时

## 核心代码逻辑

### 1. 数码管表（共阳极）
```c
char segtable[16] = {
    0xC0, 0xF9, 0xA4, 0xB0, 0x99, 0x92, 0x82, 0xF8,
    0x80, 0x90, 0x88, 0x83, 0xC6, 0xA1, 0x86, 0x8E
};
```

### 2. 字符段码映射
- 'C' 使用 'C' 的段码：0xC6
- 'U' 使用 'U' 的段码：0xC1
- 'L' 使用 'L' 的段码：0xC7
- 'R' 使用 'R' 的段码：0x88
- 'D' 使用 'D' 的段码：0xA1

### 3. 动态扫描（timer_handle）
每次T0中断输出一位：先消隐（0xFF），输出位码到GPIO_DATA，输出段码到GPIO_DATA2，pos++。

### 4. 按键处理（button_handle）
根据按键位判断是哪个键，将所有8位数码管的段码缓冲区全部设置为该字符的段码。
