# 微机原理实验备考总结

## 一、硬件平台 CPU_INT_TIMER 设备清单

| 设备 | 基地址宏 | 功能 | 方向 | 位宽 |
|------|----------|------|------|------|
| GPIO_0 通道1 | XPAR_AXI_GPIO_0_BASEADDR | 16位开关输入 | 输入 | 16bit |
| GPIO_0 通道2 | XPAR_AXI_GPIO_0_BASEADDR | 16位LED输出 | 输出 | 16bit |
| GPIO_1 通道1 | XPAR_AXI_GPIO_1_BASEADDR | 数码管位码（COM） | 输出 | 8bit |
| GPIO_1 通道2 | XPAR_AXI_GPIO_1_BASEADDR | 数码管段码（SEG） | 输出 | 8bit |
| GPIO_2 通道1 | XPAR_AXI_GPIO_2_BASEADDR | 5位按键输入 | 输入 | 5bit |
| Timer_0 T0 | XPAR_AXI_TIMER_0_BASEADDR | 定时器（1ms） | — | 32bit |
| INTC_0 | XPAR_AXI_INTC_0_BASEADDR | 中断控制器 | — | 32bit |

## 二、必须背熟的数码管段码表（共阳极）

```
数字    0     1     2     3     4     5     6     7
段码  0xC0  0xF9  0xA4  0xB0  0x99  0x92  0x82  0xF8

数字    8     9     A     b     C     d     E     F
段码  0x80  0x90  0x88  0x83  0xC6  0xA1  0x86  0x8E
```

**消隐码**：`0xFF`（所有段熄灭）

**助记口诀**（共阳极：0亮1灭）：
- 0: C0（只有中间横不亮→C0）
- 1: F9（只有右两竖亮→F9）
- 8: 80（全部亮→80）
- F: 8E（只有b段不亮→8E）

## 三、必须掌握的位码表（共阳级扫描）

```c
short poscode[8] = {0x7F, 0xBF, 0xDF, 0xEF, 0xF7, 0xFB, 0xFD, 0xFE};
//                  第0位  第1位  第2位  第3位  第4位  第5位  第6位  第7位
```

规律：`0x7F << n`（但要注意第4位开始需要额外操作，实际用查表法更安全）

## 五、中断系统初始化（标准模板 — 必须掌握）

### 5.1 中断掩码定义
```c
#define XPAR_AXI_GPIO_0_IP2INTC_IRPT_MASK 0x1  // GPIO_0（开关）中断位
#define XPAR_AXI_GPIO_2_IP2INTC_IRPT_MASK 0x2  // GPIO_2（按键）中断位
#define XPAR_AXI_TIMER_0_INTERRUPT_MASK   0x4  // Timer_0 中断位
```

### 5.2 GPIO 中断初始化（以GPIO_0开关为例）
```c
// 1. 设置方向：通道1输入，通道2输出
Xil_Out16(BASEADDR + XGPIO_TRI_OFFSET,  0xFFFF);  // 通道1：全输入
Xil_Out16(BASEADDR + XGPIO_TRI2_OFFSET, 0x0);     // 通道2：全输出

// 2. 清除中断状态
Xil_Out32(BASEADDR + XGPIO_ISR_OFFSET, XGPIO_IR_CH1_MASK);

// 3. 使能通道1中断
Xil_Out32(BASEADDR + XGPIO_IER_OFFSET, XGPIO_IR_CH1_MASK);

// 4. 全局使能GPIO中断
Xil_Out32(BASEADDR + XGPIO_GIE_OFFSET, XGPIO_GIE_GINTR_ENABLE_MASK);
```

### 5.3 GPIO_2 按键中断初始化（5位输入）
```c
Xil_Out8(BASEADDR + XGPIO_TRI_OFFSET, 0x1F);   // 低5位输入
Xil_Out32(BASEADDR + XGPIO_ISR_OFFSET, XGPIO_IR_CH1_MASK);
Xil_Out32(BASEADDR + XGPIO_IER_OFFSET, XGPIO_IR_CH1_MASK);
Xil_Out32(BASEADDR + XGPIO_GIE_OFFSET, XGPIO_GIE_GINTR_ENABLE_MASK);
```

### 5.4 定时器初始化（1ms周期）
```c
#define RESET_VALUE0  1000 - 2  // 1ms计数值（100MHz时钟，100000*0.00001=1000）

// 1. 停止定时器
int csr = Xil_In32(BASEADDR + XTC_TCSR_OFFSET);
Xil_Out32(BASEADDR + XTC_TCSR_OFFSET, csr & ~XTC_CSR_ENABLE_TMR_MASK);

// 2. 设置初值
Xil_Out32(BASEADDR + XTC_TLR_OFFSET, RESET_VALUE0);

// 3. 加载初值
Xil_Out32(BASEADDR + XTC_TCSR_OFFSET, csr | XTC_CSR_LOAD_MASK);

// 4. 配置并启动：递减 + 自动重载 + 中断使能 + 清除中断标志 + 使能定时器
Xil_Out32(BASEADDR + XTC_TCSR_OFFSET,
    (csr & ~XTC_CSR_LOAD_MASK) |
    XTC_CSR_ENABLE_INT_MASK |      // 使能中断
    XTC_CSR_AUTO_RELOAD_MASK |     // 自动重载
    XTC_CSR_DOWN_COUNT_MASK |      // 递减计数
    XTC_CSR_INT_OCCURED_MASK |     // 清除中断标志（写1清0）
    XTC_CSR_ENABLE_TMR_MASK);      // 使能定时器
```

### 5.5 中断控制器初始化
```c
// 1. 清除所有待处理中断
Xil_Out32(INTC_BASEADDR + XIN_IAR_OFFSET,
    GPIO_0_MASK | GPIO_2_MASK | TIMER_MASK);

// 2. 使能中断源
Xil_Out32(INTC_BASEADDR + XIN_IER_OFFSET,
    GPIO_0_MASK | GPIO_2_MASK | TIMER_MASK);

// 3. 主使能（硬件中断 + 主使能）
Xil_Out32(INTC_BASEADDR + XIN_MER_OFFSET,
    XIN_INT_MASTER_ENABLE_MASK | XIN_INT_HARDWARE_ENABLE_MASK);

// 4. 使能MicroBlaze全局中断
microblaze_enable_interrupts();
```

## 六、中断服务函数 ISR（普通中断 — 必须掌握）

### 6.1 ISR 声明
```c
void My_ISR() __attribute__((interrupt_handler));
```

### 6.2 ISR 实现（标准模式）
```c
void My_ISR()
{
    // 1. 读取中断状态寄存器
    int status = Xil_In32(XPAR_AXI_INTC_0_BASEADDR + XIN_ISR_OFFSET);

    // 2. 逐一判断中断源并调用处理函数
    if ((status & XPAR_AXI_GPIO_0_IP2INTC_IRPT_MASK) == XPAR_AXI_GPIO_0_IP2INTC_IRPT_MASK)
        switch_handle();
    if ((status & XPAR_AXI_GPIO_2_IP2INTC_IRPT_MASK) == XPAR_AXI_GPIO_2_IP2INTC_IRPT_MASK)
        button_handle();
    if ((status & XPAR_AXI_TIMER_0_INTERRUPT_MASK) == XPAR_AXI_TIMER_0_INTERRUPT_MASK)
        timer_handle();

    // 3. 清除中断（写1清0 — 必须！）
    Xil_Out32(XPAR_AXI_INTC_0_BASEADDR + XIN_IAR_OFFSET, status);
}
```

## 七、核心功能C语言逻辑 — 必须掌握

### 7.1 开关读取（switch_handle 模板）
```c
void switch_handle()
{
    short sw = Xil_In16(XPAR_AXI_GPIO_0_BASEADDR + XGPIO_DATA_OFFSET);
    Xil_Out16(XPAR_AXI_GPIO_0_BASEADDR + XGPIO_DATA2_OFFSET, sw);  // LED实时
    display_val = sw;
    // 清除GPIO中断
    Xil_Out16(BASEADDR + XGPIO_ISR_OFFSET, Xil_In16(BASEADDR + XGPIO_ISR_OFFSET));
}
```

### 7.2 按键读取与消抖（button_handle 模板）
```c
void button_handle()
{
    char btn = Xil_In8(XPAR_AXI_GPIO_2_BASEADDR + XGPIO_DATA_OFFSET) & 0x1F;
    if (btn == 0) {  // 无按键按下
        Xil_Out32(BASEADDR + XGPIO_ISR_OFFSET, Xil_In32(BASEADDR + XGPIO_ISR_OFFSET));
        return;
    }
    // 按键位定义：
    // bit0(0x01) = C键
    // bit1(0x02) = U键
    // bit2(0x04) = L键
    // bit3(0x08) = R键
    // bit4(0x10) = D键
    // 注意：按键按下为高电平
    if (btn & 0x01) { /* C键处理 */ }
    if (btn & 0x02) { /* U键处理 */ }
    // ... 清除中断
}
```

### 7.3 数码管动态扫描（timer_handle 核心模板）
```c
void timer_handle()
{
    // 1. 消隐（先关段码再关位码，防止鬼影）
    Xil_Out8(GPIO1_BASE + XGPIO_DATA_OFFSET, 0xFF);   // 关闭所有位
    Xil_Out8(GPIO1_BASE + XGPIO_DATA2_OFFSET, 0xFF);  // 关闭所有段

    // 2. 更新段码（if需要需要：根据模式更新segcode数组）
    // segcode[i] = segtable[...];

    // 3. 输出当前位
    Xil_Out8(GPIO1_BASE + XGPIO_DATA_OFFSET, poscode[pos]);  // 选通位
    Xil_Out8(GPIO1_BASE + XGPIO_DATA2_OFFSET, segcode[pos]); // 输出段码

    // 4. 位指针递增
    pos++;
    if (pos == 8) pos = 0;  // 8位数码管循环

    // 5. 清除定时器中断标志（写1清0 — 必须！）
    Xil_Out32(TIMER_BASE + XTC_TCSR_OFFSET,
        Xil_In32(TIMER_BASE + XTC_TCSR_OFFSET) | XTC_CSR_INT_OCCURED_MASK);
}
```

### 7.4 十六进制拆分显示（最常用）
```c
// 将16位值拆分为4个4位十六进制数
unsigned short v = display_val;
for (int i = 0; i < 4; i++)
    segcode[i] = segtable[(v >> (12 - 4*i)) & 0xF];
// 后4位消隐
for (int i = 4; i < 8; i++)
    segcode[i] = 0xFF;
```

## 十、常见BUG与注意事项

### 10.1 中断清除（最容易出错）
- INTC中断清除：`Xil_Out32(INTC_BASE + XIN_IAR_OFFSET, status)` — 写1清0
- Timer中断清除：`Xil_Out32(TIMER_BASE + XTC_TCSR_OFFSET, csr | XTC_CSR_INT_OCCURED_MASK)` — 写1清0
- GPIO中断清除：`Xil_Out32(GPIO_BASE + XGPIO_ISR_OFFSET, Xil_In32(GPIO_BASE + XGPIO_ISR_OFFSET))` — 读后写回

### 10.2 消隐顺序
必须先关位码（COM）再改段码，否则会出现"鬼影"。

### 10.3 按键消抖
按键中断处理中要检查 `btn == 0` 直接返回，防止噪声触发。

### 10.4 数组越界
`pos` 从0到7，共8位，`pos == 8` 时归零。

### 10.5 中断声明
ISR函数必须有 `__attribute__((interrupt_handler))`，否则中断处理中寄存器不会被正确保存/恢复。
