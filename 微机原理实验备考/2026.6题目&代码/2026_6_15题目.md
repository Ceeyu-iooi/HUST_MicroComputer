# 2026.6.15 期末考试题目

## 题目描述

嵌入式计算机系统将独立按键作为输入设备，LED 灯[7:0]作为输出设备。

### a. 按键 BTNC 功能

点按键 **BTNC**：低八位 LED 灯[7:0] 显示 `1111 0000`（按下后才显示），其余LED灯熄灭。

### b. 按键 BTNL 功能（三态循环）

点按键 **BTNL**，按一次切换一种状态，三种状态循环切换：

| 按键次数 mod 3 | 状态 | 功能说明 |
|:---:|:---:|---|
| 0（按下即进入） | **左移状态** | LED灯整体以 **1Hz** 自动左移一位，到最左边的时候回到最低位（循环左移）<br>例如：`1111 0000` → `1110 0001` → `1100 0011` → ... |
| 1（再按一次） | **右移状态** | LED灯整体以 **1Hz** 自动右移一位，到最右边的时候回到最左边（循环右移）<br>例如：`1111 0000` → `0111 1000` → `0011 1100` → ... |
| 2（再按一次） | **停止状态** | LED灯停止移动，保持当前 LED 值不变（冻结显示） |

> 再按一次 BTNL 回到左移状态，依此循环。

### 要求

- **a. 中断方式实现**：采用普通中断方式实现所有功能。
  - 向左循环时，到最左边的时候回到最低位；
  - 向右循环时，到最右边的时候回到最高位。
- **b. 实验报告要求**：
  - 画出硬件原理框图
  - 软件算法、模块组成以及模块之间的接口
  - 软件流程图

## 功能分析

### 按键 BTNC 功能分析

- 触发方式：按下 BTNC 瞬间立即响应
- LED 输出：低 8 位显示 `0xF0`（`1111 0000`），高 8 位 LED 熄灭
- 执行位置：按键中断服务函数中直接操作

### 按键 BTNL 三态循环分析

使用计数器 `count` 记录 BTNL 按下次数，`count % 3` 决定当前状态：

```
count % 3 == 0 → 左移模式（MODE_L1）
count % 3 == 1 → 右移模式（MODE_L2）
count % 3 == 2 → 停止模式（MODE_L3）
```

三种状态均在 **T1 定时器中断** 中执行（1Hz = 1秒一次），由 `mode` 全局变量控制：

| mode 值 | 状态 | T1 定时器中的操作 |
|:---:|:---:|---|
| `MODE_L1` (3) | 左移 | `Led_current = ((Led_current<<1) & 0xFF) \| ((Led_current>>7) & 0x01)` |
| `MODE_L2` (4) | 右移 | `Led_current = ((Led_current<<7) & 0x80) \| ((Led_current>>1) & 0xFF)` |
| `MODE_L3` (5) | 停止 | 不做任何操作，LED 保持当前值 |

### 移位说明

**左移（循环左移）：**

```
初始：  1111 0000  (0xF0)
第1秒： 1110 0001  (0xE1)  ← 最高位移出，补到最低位
第2秒： 1100 0011  (0xC3)
第3秒： 1000 0111  (0x87)
...     循环继续
```

**右移（循环右移）：**

```
初始：  1111 0000  (0xF0)
第1秒： 0111 1000  (0x78)  ← 最低位移出，补到最高位
第2秒： 0011 1100  (0x3C)
第3秒： 0001 1110  (0x1E)
...     循环继续
```

## 硬件平台

| 外设 | 说明 |
|------|------|
| AXI GPIO 0 | CH1=16位独立开关（输入），CH2=16位LED灯（输出） |
| AXI GPIO 1 | CH1=8位数码管位选（输出），CH2=8位数码管段码（输出） |
| AXI GPIO 2 | CH1=5位独立按键（输入），支持中断 |
| AXI Timer 0 | T0通道=数码管动态扫描定时器（约10μs），T1通道=1Hz节奏定时器（1s） |
| 中断控制器 | GPIO2按键中断（IRQ_MASK=0x2）+ Timer0中断（IRQ_MASK=0x4） |

## 中断配置

| 中断源 | 中断掩码 | 功能 | 中断周期 |
|--------|---------|------|---------|
| Timer0 (T0) | 0x4 | 数码管动态扫描 | 约10μs |
| Timer1 (T1) | 0x4（与T0共用同一根中断线） | 1Hz LED移位节奏控制 | 1s |
| GPIO2 (按键) | 0x2 | 按键检测（BTNC/BTNL） | 按键按下/松开时触发 |

## 考试注意事项

1. **BTNC** 按下即显示 `0xF0`，松开不影响 LED（松开中断仅清标志）
2. **BTNL** 三种状态循环，用 `count % 3` 区分，初始 `count=0` 时按下即进入左移
3. **T1 定时器** 重装载值 = `100000000 - 2`（100MHz → 1Hz）
4. **普通中断**：统一 ISR 入口 `My_ISR()`，软件读取 INTC ISR 寄存器分发
5. 左移/右移均为 **循环移位**（移出位补到另一端）
