### 硬件连接说明

| 信号路径 | 说明 |
|---------|------|
| BTNC → GPIO2_CH1 bit0 | 中间按键作为输入，按下时电平变化触发中断 |
| BTNL → GPIO2_CH1 bit2 | 左键作为输入，触发中断控制三态循环 |
| GPIO2 中断 → INTC IRQ1 | 按键中断请求连接到中断控制器中断源1（掩码0x2） |
| Timer0 T0/T1 中断 → INTC IRQ2 | 定时器中断共享中断源2（掩码0x4） |
| GPIO0_CH2 → LED[15:0] | 16位LED输出，本实验仅用低8位LED[7:0] |
| GPIO1_CH1 → 位选[7:0] | 8位数码管位选，低电平有效 |
| GPIO1_CH2 → 段码[7:0] | 8位数码管段码（共阳极） |

## 二、软件算法设计

### 2.1 整体算法流程

```c
main()
├── 初始化 GPIO 方向
├── 使能 GPIO2 按键中断
├── 初始化 T0 定时器（10μs 数码管扫描）
├── 初始化 T1 定时器（1s LED 移位节奏）
├── 初始化 INTC 中断控制器
├── 使能 CPU 全局中断
├── LED 全灭
└── while(1) 等待中断
```

所有功能由中断驱动，主循环为空循环。

### 2.2 按键 BTNC 算法

```
BTNC 按下
├── mode = MODE_C
├── 数码管全灭
├── 输出 LED = 0x00F0（低8位=1111 0000，高8位=0）
└── Led_current = 0xF0（同步更新，供后续 BTNL 使用）
```

### 2.3 按键 BTNL 三态循环算法

```
BTNL 按下
├── if (count == 0) → Led_current = 0xF0（首次按下初始化）
│
├── if (count % 3 == 0) → MODE_L1（左移模式）
│       T1 每 1s 执行: Led_current 循环左移1位
│       count++
│
├── else if (count % 3 == 1) → MODE_L2（右移模式）
│       T1 每 1s 执行: Led_current 循环右移1位
│       count++
│
└── else if (count % 3 == 2) → MODE_L3（停止模式）
        LED 保持当前值，T1 不做操作
        count++
```

### 2.4 循环移位算法

**循环左移**（MODE_L1）：

```
Led_current = ((Led_current << 1) & 0xFF) | ((Led_current >> 7) & 0x01)

图解：
  MSB(bit7)                      LSB(bit0)
  ┌──────────────────────────────┐
  │  1  1  1  1  0  0  0  0     │ 初始: 0xF0
  └──────────────────────────────┘
         ↙ 左移1位，bit7移出 → 补到bit0
  ┌──────────────────────────────┐
  │  1  1  1  0  0  0  0  1     │ 结果: 0xE1
  └──────────────────────────────┘
```

**循环右移**（MODE_L2）：

```
Led_current = ((Led_current << 7) & 0x80) | ((Led_current >> 1) & 0xFF)

图解：
  MSB(bit7)                      LSB(bit0)
  ┌──────────────────────────────┐
  │  1  1  1  1  0  0  0  0     │ 初始: 0xF0
  └──────────────────────────────┘
         ↙ 右移1位，bit0移出 → 补到bit7
  ┌──────────────────────────────┐
  │  0  1  1  1  1  0  0  0     │ 结果: 0x78
  └──────────────────────────────┘
```

## 三、模块组成及接口

### 3.1 模块划分

| 模块 | 函数名 | 功能描述 |
|------|--------|---------|
| **系统初始化** | `Initialization()` | 配置所有GPIO方向、使能中断、初始化T0/T1定时器、使能INTC、使能CPU中断 |
| **主中断入口** | `My_ISR()` | 统一中断入口，读取INTC ISR寄存器，分发到按键/定时器处理函数 |
| **定时器分发** | `timer_handle()` | 读取T0/T1 TCSR寄存器，区分T0和T1的中断 |
| **T0处理** | `timer0_handle()` | 数码管动态扫描（10μs/位），输出位选+段码 |
| **T1处理** | `timer1_handle()` | 1Hz LED移位：MODE_L1左移 / MODE_L2右移 / MODE_L3停止 |
| **按键处理** | `button_handle()` | 读取按键值：BTNC→显示0xF0，BTNL→三态循环切换 |

### 3.2 模块间接口

```
┌─────────────────────────────────────────────────────────────────┐
│                          全局状态变量                              │
│  mode (int)        : 当前显示模式，控制T1行为                    │
│  count (int)       : BTNL按下次数，控制三态循环                  │
│  Led_current (char): LED当前值，左移/右移操作对象                │
│  segcode[8] (char) : 数码管段码缓冲区                           │
│  pos (int)         : 数码管动态扫描当前位置                     │
└─────────────────────────────────────────────────────────────────┘
          ↑                      ↑                      ↑
          │                      │                      │
    ┌─────┴──────┐        ┌─────┴──────┐        ┌──────┴─────┐
    │button_handle│        │timer0_handle│        │timer1_handle│
    │ (写mode,    │        │ (读segcode, │        │ (读mode,    │
    │  count,     │        │  poscode,   │        │  读/写      │
    │  Led_current)│        │  pos)      │        │  Led_current)│
    └─────────────┘        └────────────┘        └────────────┘
```

### 3.3 全局状态变量

| 变量名 | 类型 | 初始值 | 作用域 | 功能 |
|--------|------|--------|--------|------|
| `mode` | `int` | `MODE_IDLE` | 全局 | 控制T1的移位行为 |
| `count` | `int` | `0` | 全局 | BTNL按下次数计数器 |
| `Led_current` | `char` | `0` | 全局 | 当前LED显示值，移位操作对象 |
| `segcode[8]` | `char[]` | 全`0xff` | 全局 | 8位数码管段码缓冲区 |
| `pos` | `int` | `0` | 全局 | 动态扫描当前位置（0~7） |

## 五、关键代码说明

### 5.1 循环左移实现

```c
/* 循环左移：Led_current 整体左移1位，最高位移出补到最低位 */
Led_current = ((Led_current << 1) & 0x00FF) | ((Led_current >> 7) & 0x0001);
Xil_Out16(XPAR_AXI_GPIO_0_BASEADDR + XGPIO_DATA2_OFFSET, Led_current & 0x00FF);
```

- `(Led_current << 1)`：整体左移1位，bit0补0
- `& 0x00FF`：屏蔽高8位，只保留低8位
- `(Led_current >> 7) & 0x0001`：原最高位(bit7)提取到bit0位置
- 两者相或：实现循环左移

### 5.2 循环右移实现

```c
/* 循环右移：Led_current 整体右移1位，最低位移出补到最高位 */
Led_current = ((Led_current << 7) & 0x0080) | ((Led_current & 0x00FF) >> 1);
Xil_Out16(XPAR_AXI_GPIO_0_BASEADDR + XGPIO_DATA2_OFFSET, Led_current & 0x00FF);
```

- `(Led_current << 7) & 0x0080`：原最低位(bit0)左移到bit7位置
- `(Led_current & 0x00FF) >> 1`：整体右移1位，bit7补0
- 两者相或：实现循环右移

### 5.3 BTNL 三态切换逻辑

```c
/* count % 3 决定状态 */
if (count == 0) Led_current = 0xF0;         /* 首次按下初始化 */

if (count % 3 == 0)      /* 左移模式 */
    mode = MODE_L1;
else if (count % 3 == 1) /* 右移模式 */
    mode = MODE_L2;
else if (count % 3 == 2) /* 停止模式 */
{
    mode = MODE_L3;
    Xil_Out16(..., Led_current & 0x00FF);     /* 停止时显示当前值 */
}
count++;
```

- `count` 从0开始递增，永不重置，循环周期为3
- `count % 3 == 0`：初始状态，左移
- `count % 3 == 1`：右移
- `count % 3 == 2`：停止（冻结显示）
