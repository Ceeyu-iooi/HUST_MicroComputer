# 题目2：开关计数器（程序控制方式）

## 题目要求
- 程序读取独立开关值作为输入
- 按键按下时，读取当前开关值作为计数器初始值
- 数码管显示计数值（0~255）

## 硬件资源
| 外设 | GPIO | 方向 |
|------|------|------|
| 独立开关(16位) | GPIO0_CH1 | 输入 |
| 数码管段码(8位) | GPIO1_CH1 | 输出 |
| 数码管位选(8位) | GPIO1_CH2 | 输出 |
| 独立按键(5位) | GPIO2_CH1 | 输入 |

## 核心实现逻辑

### 1. 程序控制方式（无中断）
使用**轮询**方式检测按键：
```c
while(1) {
    unsigned char key = Xil_In8(GPIO2_BASE + XGPIO_DATA_OFFSET) & 0x1F;
    if (key) {
        // 读取开关值，作为计数初值
        count_value = Xil_In16(GPIO0_BASE + XGPIO_DATA_OFFSET) & 0xFF;
        // 更新数码管显示
        update_display(count_value);
    }
}
```

### 2. 数码管显示
- 使用动态扫描方式显示8位数字
- 通常显示低2位或4位数字
- 需要定时器或软件延时配合扫描

### 3. 计数值处理
- 每次按键读取开关值作为新计数
- 不做累加，每次重新赋值
- 仅取低8位（0~255）

## 关键点
- 程序控制：无中断，纯轮询
- GPIO方向配置：开关输入、数码管输出、按键输入
- 数码管动态扫描需要定时器或延时
