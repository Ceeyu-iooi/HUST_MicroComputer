# Test8：8位7段数码管每秒滚动显示数字序列3456

## 功能概述

在8位7段共阳极数码管上，每秒将数字序列 **"3456"** 向左滚动一位，没有数字的位置全部熄灭（0xff）。初始状态时，3456 显示在最右边4位数码管上，随后每秒向左移动一位，形成循环滚动效果。

## 硬件平台

| 外设 | 说明 |
|------|------|
| AXI GPIO 0 | CH1=16位独立开关（输入），CH2=16位LED灯（输出，本测试未使用） |
| AXI GPIO 1 | CH1=8位数码管位选（输出），CH2=8位数码管段码（输出） |
| AXI Timer 0 | T0通道=数码管动态扫描定时器（约10μs），T1通道=滚动节奏定时器（1s） |
| 中断控制器 | GPIO2（按键）中断使能但未使用，Timer0 中断用于T0/T1 |

## 中断配置

| 中断源 | 中断掩码 | 功能 | 中断周期 |
|--------|---------|------|---------|
| Timer0 (T0) | 0x4 | 数码管动态扫描 | 约10μs |
| Timer1 (T1) | 0x4（与T0共用同一根中断线） | 滚动控制 | 1s |
| GPIO2 (按键) | 0x2 | 保留（本测试未使用） | — |

> **注意**：T0和T1是同一个AXI Timer IP核的两个独立通道，共享同一根中断线进入中断控制器。在 `My_ISR` 中收到中断后，需通过 `timer_handle()` 分发函数分别读取两个通道的TCSR寄存器，判断是哪个通道触发的中断。

## 数码管动态扫描原理

1. T0 定时器每约 **10μs** 触发一次中断
2. 在 `timer0_handle()` 中：
   - **消影**：先关闭所有位选和段码，防止切换瞬间显示残影
   - 输出当前位（`pos`）的位选码和对应段码
   - 切换到下一位（`pos++`，到8归0）
3. 因刷新频率极高（约12500次/秒），人眼视觉暂留效应使8位数码管看起来同时稳定点亮

## 滚动逻辑

### 初始状态（t=0）

```
数码管位置：[最左]    [2]    [3]    [4]    [5]    [6]    [7]  [最右]
segcode:    [ff]   [ff]   [ff]   [ff]   [3]    [4]    [5]    [6]
显示：        灭     灭     灭     灭     3      4      5      6
```

### 滚动过程

每1秒（T1中断触发），`timer1_handle()` 执行一次循环左移：

```
t=0： [ff][ff][ff][ff][3][4][5][6]   ← 初始状态
t=1： [ff][ff][ff][3][4][5][6][ff]   ← 向左移1位，3移出进入最右边？
t=2： [ff][ff][3][4][5][6][ff][ff]
t=3： [ff][3][4][5][6][ff][ff][ff]
t=4： [3][4][5][6][ff][ff][ff][ff]   ← 4秒后3456移到最左边
t=5： [4][5][6][ff][ff][ff][ff][3]   ← 3循环到最右边
t=6： [5][6][ff][ff][ff][ff][3][4]
t=7： [6][ff][ff][ff][ff][3][4][5]
t=8： [ff][ff][ff][ff][3][4][5][6]   ← 8秒后回到初始状态
...循环...
```

### 循环左移代码逻辑

```c
void timer1_handle(void)
{
    char temp = segcode[0];              // 保存最左端(第1位)的值
    for (int i = 0; i < 7; i++)
    {
        segcode[i] = segcode[i + 1];     // 后一位覆盖前一位（向左移）
    }
    segcode[7] = temp;                   // 原最左端值移到最右端(第8位)

    /* 清除 T1 中断标志 */
    Xil_Out32(XPAR_AXI_TIMER_0_BASEADDR + XTC_TIMER_COUNTER_OFFSET + XTC_TCSR_OFFSET,
              Xil_In32(...) | XTC_CSR_INT_OCCURED_MASK);
}
```

## 关键寄存器配置

### T0 定时器（数码管扫描）
| 寄存器 | 初始值 | 说明 |
|--------|--------|------|
| TLR0 | 1000 - 2 | 约10μs（100MHz时钟） |
| TCSR0 | 使能中断 + 自动重载 + 减计数 + 使能 | 持续产生定时中断 |

### T1 定时器（滚动节奏）
| 寄存器 | 初始值 | 说明 |
|--------|--------|------|
| TLR1 | 100000000 - 2 | 约1s（100MHz时钟） |
| TCSR1 | 使能中断 + 自动重载 + 减计数 + 使能 | 每秒触发一次滚动 |

### 中断控制器
| 寄存器 | 写入值 | 说明 |
|--------|--------|------|
| IAR | GPIO_2_IRQ_MASK \| TIMER_0_IRQ_MASK | 清除按键和定时器中断标志 |
| IER | GPIO_2_IRQ_MASK \| TIMER_0_IRQ_MASK | 使能按键和定时器中断 |
| MER | Master Enable \| Hardware Enable | 中断控制器全局使能 |

## 中断响应流程图

```
Timer0溢出(T0,~10μs)         Timer0溢出(T1,1s)
        ↓                          ↓
  中断控制器(INTC) ←──────── 共享中断线(TIMER_0_IRQ_MASK=0x4)
        ↓
   My_ISR()
        ↓
  timer_handle()
    ├── 读T0.TCSR → 有INT_OCCURED? → timer0_handle()
    │                                      ├── 消影
    │                                      ├── 输出当前位选+段码
    │                                      ├── pos++（循环0~7）
    │                                      └── 清T0中断标志
    └── 读T1.TCSR → 有INT_OCCURED? → timer1_handle()
                                           ├── 循环左移segcode
                                           └── 清T1中断标志
```

## Vitis 工程配置要点

1. **AXI Timer**：必须将定时器配置为 **2个定时器通道**（Generate 2 Timers），否则无法使用T1
2. **中断控制器**：确保 Timer0 的中断已连接到中断控制器的 **中断源2**（对应掩码0x4）
3. **GPIO2**：即使未使用按键，初始化了其中断。`button_handle()` 为空函数，仅清除中断标志，避免误触发时死循环

## 常见问题

**Q：为什么数码管显示闪烁？**
A：T0 定时器的 RESET_VALUE0 过大会导致刷新率太低。确保 TLR0 = 1000 - 2（约10μs），而非 100000000（1s）。

**Q：为什么数码管不显示任何内容？**
A：检查（1）GPIO1的TRI方向寄存器是否配置为输出（写0）；（2）segtable_hex[3~6] 是否对应正确的段码；（3）poscode 位选码是否低电平有效。

**Q：数字滚动速度不对？**
A：T1 定时器的 RESET_VALUE1 = 100000000 - 2 对应 100MHz 时钟的 1 秒。如果系统时钟不同，需按实际时钟频率计算重装载值。
