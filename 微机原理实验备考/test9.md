# Test9: 5按键多功能系统（2个定时器实现）

## 功能概述

基于CPU_INT_TIMER硬件平台（MicroBlaze + N4DDR），用2个定时器中断实现5种显示模式，通过5个独立按键（C/U/L/R/D）切换：

| 按键 | 模式 | 功能描述 |
|------|------|----------|
| **C** | 模式C | 数码管全灭，流水灯步进（1<<ledbits, ledbits=0~15循环）。首次按=0.5s，再按=0.25s，循环 |
| **U** | 模式U | 数码管右2位显示开关低4位的原码十进制（负号+绝对值），T1驱动循环左移滚动。首次按=1s，再按=0.5s，循环。LED反映开关值 |
| **L** | 模式L | 首次按：LED显示16位开关值；再按：LED逻辑左移1位（高位丢弃，低位补0）。16次后复位 |
| **R** | 模式D | 数码管右4位显示开关低4位的二进制反码（0/1），左4位熄灭。LED偶数位亮(0xAAAA)、奇数位灭 |
| **D** | 模式C | 数码管左4位显示开关低4位的二进制（0/1），右4位熄灭。LED奇数位亮(0x5555)、偶数位灭 |

## 硬件配置

- **平台**：CPU_INT_TIMER（MicroBlaze最小系统 + N4DDR）
- **外设**：
  - AXI GPIO0：CH1=16位独立开关（输入）, CH2=16位LED灯（输出）
  - AXI GPIO1：CH1=8位数码管位选（输出）, CH2=8位数码管段码（输出）
  - AXI GPIO2：CH1=5位独立按键（输入，中断方式）
  - AXI Timer0：双通道定时器，T0=数码管扫描（10μs），T1=周期性操作基频
  - AXI INTC：中断控制器

## 定时器分配

### T0定时器（10μs中断）
- **装载值**：`1000 - 2`（100MHz时钟）
- **功能**：数码管动态扫描，每次中断切换一位，8位循环
- **中断频率**：约10μs/位，每秒刷新约12500次

### T1定时器（基础周期=1s，可动态切换速度）
- **装载值**：`100000000 - 2`（100MHz时钟，1秒）
- **功能**：产生周期性操作的基础时基，通过按键动态修改装载值切换速度
- **速度映射**：
  - `T1_BASE_TICK`（100M-2）= 1s（模式U默认）
  - `T1_TICK_halfS`（50M-2）= 0.5s（模式C默认/模式U切换）
  - `T1_TICK_quarterS`（25M-2）= 0.25s（模式C切换）

## 按键位掩码与判断顺序

GPIO2_CH1低5位对应5个独立按键。实际硬件bit映射：

| 按键 | 掩码 | Bit位 | PCB位置 |
|------|------|-------|---------|
| C    | 0x01 | bit0  | 中间键 |
| U    | 0x02 | bit1  | 上键   |
| L    | 0x04 | bit2  | 左键   |
| R    | 0x08 | bit3  | 右键   |
| D    | 0x10 | bit4  | 下键   |

`button_handle()`中使用`if-else`链按 **C → U → L → R → D** 顺序判断：
```
BTNC(bit0) → BTNU(bit1) → BTNL(bit2) → BTNR(bit3) → BTND(bit4)
```

## 各模式详细设计

### 模式C：流水灯（BTNC, bit0）

```
按键C → 进入模式C，数码管全灭，流水灯0.5s步进
  再按C → 切换为0.25s步进
  再按C → 切回0.5s步进（循环）
```

- **显示**：数码管全灭（`segcode[0~7] = 0xff`）
- **LED**：`ledbits`从0~15循环，每次输出`1 << ledbits`点亮对应LED
- **速度控制**：`count`计数器（0/1循环）
  - `count==0`：重装载值=T1_TICK_halfS（0.5s）
  - `count==1`：重装载值=T1_TICK_quarterS（0.25s）

### 模式U：原码十进制滚动（BTNU, bit1）

```
首次按U → 数码管右2位显示低4位开关的原码十进制值，1s滚动
  再按U → 切换为0.5s滚动
  再按U → 切回1s滚动（循环）
```

- **原码解析**：bit3=符号位，bit2~bit0=绝对值
  - `0000~0111` → 0~7（正数）
  - `1000` → 显示"0"（-0=0）
  - `1001~1111` → 负号 + 绝对值（-1~-7）
- **滚动机制**：T1中断中`segcode[0~7]`整体循环左移1位
- **速度控制**：`count1`计数器（0/1循环）
  - `count1==0`：重装载值=T1_BASE_TICK（1s）
  - `count1==1`：重装载值=T1_TICK_halfS（0.5s）
- **LED**：反映开关低4位值

### 模式L：LED=开关 + 左移（BTNL, bit2）

```
首次按L → LED=16位开关值，数码管显示低4位二进制
  再按L → LED逻辑左移1位（高位丢弃，低位补0）
  再按L → 继续左移...
  16次后复位，下次按L重新读取开关值
```

- **LED**：首次按读取开关值存入`sw_current`，后续按`sw_current = (sw_current & 0x7fff) | ((sw_current >> 15) & 0x0001)`逻辑左移
- **数码管**：通过T1中断中`segcode`循环左移实现滚动（模式U独有，模式L无自动滚动）
- **左移计数器**：`count2`（0~15），满16后归0

### 模式D（R键）：反码 + 偶数LED（BTNR, bit3）

```
按R → 数码管右4位=开关低4位二进制反码（0/1显示）
      左4位熄灭
      LED偶数位亮（0xAAAA）
```

- **数码管右4位**：`segcode[4~7]`显示`~low4`的每位（0→显示"1"，1→显示"0"）
- **LED**：输出固定值`0xAAAA`（1010 1010 1010 1010），偶数位亮、奇数位灭

### 模式C（D键）：二进制 + 奇数LED（BTND, bit4）

```
按D → 数码管左4位=开关低4位二进制（0/1显示）
      右4位熄灭
      LED奇数位亮（0x5555）
```

- **数码管左4位**：`segcode[0~3]`显示开关低4位的每位（1→显示"1"，0→显示"0"）
- **LED**：输出固定值`0x5555`（0101 0101 0101 0101），奇数位亮、偶数位灭

## 各模式LED/数码管行为汇总

| 模式 | 按键 | LED行为 | 数码管行为 |
|------|------|---------|------------|
| C    | C    | 流水灯（1<<ledbits, ledbits=0~15循环）| 全灭 |
| U    | U    | 显示开关低4位（low4） | [熄灭×6][负号][数值]，T1驱动滚动 |
| L    | L    | 首次=开关值，再按逻辑左移 | 显示低4位二进制（T0扫描） |
| D(R) | R    | 固定0xAAAA（偶数位亮）| [熄灭×4][反码×4] |
| C(D) | D    | 固定0x5555（奇数位亮）| [二进制×4][熄灭×4] |

## 中断架构

```
My_ISR()                          // 主中断入口
├── button_handle()               // GPIO2按键中断（优先级高）
│   ├── BTNC(bit0) → mode=MODE_C // 流水灯
│   ├── BTNU(bit1) → mode=MODE_U // 十进制滚动
│   ├── BTNL(bit2) → mode=MODE_L // LED=开关+左移
│   ├── BTNR(bit3) → mode=MODE_R // 反码+偶数LED
│   └── BTND(bit4) → mode=MODE_D // 二进制+奇数LED
│
└── timer_handle()                // 定时器中断
    ├── timer0_handle()           // T0: 数码管动态扫描（10μs/位）
    └── timer1_handle()           // T1: 周期性操作
        ├── MODE_C → 流水灯步进
        ├── MODE_U → segcode循环左移
        └── 其他 → 无操作
```

## 关键技术点

1. **双定时器协同**：T0负责高频数码管扫描（10μs），T1提供低频基础周期（1s），通过动态修改装载值切换速度
2. **按键中断**：GPIO2按键中断优先级高于定时器，确保按键响应及时
3. **if-else链**：C→U→L→R→D顺序判断，高优先级按键覆盖低优先级
4. **防抖处理**：按键按下时触发一次操作，松开时只清除中断标志
5. **段码复用**：`segtable_hex[0/1]`既能显示数字0~F，也能显示0/1的二进制表示
6. **流水灯实现**：`1 << ledbits`（ledbits=0~15）实现单LED流动
